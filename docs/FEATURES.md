## 2. AKIH-Methodologie

### Was ist AKIH?
AI-Assisted Key Insight Harvesting - eine wissenschaftlich fundierte Methode zur qualitativen Datenanalyse.

### Phasen
1. **Initiale Codierung**: Erste Durchsicht und Markierung
2. **Kategorisierung**: Gruppierung verwandter Codes
3. **Themenfindung**: Identifikation übergeordneter Muster
4. **Validierung**: Überprüfung durch Multi-Experten-System

## 9. Fehlerbehebung

### App startet nicht
1. Cache löschen: `%APPDATA%/EVIDENRA BASIC`
2. Als Administrator starten
3. Antivirus-Ausnahme hinzufügen

### Lizenz-Probleme
1. Internetverbindung prüfen
2. Lizenz erneut eingeben
3. Support kontaktieren

### Performance
1. Große PDFs in Teilen verarbeiten
2. Nicht benötigte Dokumente entfernen
3. App neu starten
